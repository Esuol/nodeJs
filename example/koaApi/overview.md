## Koa

### introduction

Koa是Express团队设计的一个新的web框架，它的目标是为web应用程序和api提供一个更小、更富表现力、更健壮的基础。通过利用异步函数，Koa允许您放弃回调并大大增加错误处理。Koa在其核心中不捆绑任何中间件，并且它提供了一套优雅的方法，使编写服务器变得快速和令人愉快。

### installation

Koa要求节点v7.6.0或更高版本才能支持ES2015和异步功能。

您可以使用喜欢的版本管理器快速安装受支持的节点版本：

```sh
$ nvm install 7
$ npm i koa
$ node my-koa-app.js
```
